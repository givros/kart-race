# Kart Race

Prototype de course karting Three.js avec lobby multijoueur WebSocket.

## Jouer depuis l'exterieur

Sur l'ordinateur qui heberge la partie :

```powershell
cd E:\Dev\karting
npm run public
```

Puis les joueurs ouvrent simplement :

```text
https://givros.github.io/kart-race/
```

Le terminal doit rester ouvert pendant la session. Quand tu fermes le terminal, la partie publique n'est plus accessible.

## Local

```powershell
npm run server
```

Puis ouvre :

```text
http://127.0.0.1:8787
```
